# DEV-On

Projeto estatico em HTML e CSS com tres paginas (Home, HTML e CSS) que simulam um mini blog/portal de estudos de desenvolvimento web. Inclui layout em duas colunas, formulario de comentarios e conteudo introdutorio sobre JavaScript, HTML e CSS.

## Paginas
- `index.html`: Home com artigo sobre JavaScript.
- `dev-on.html`: Pagina sobre HTML e HTML semantico.
- `css.html`: Pagina sobre a historia do CSS.

## Estrutura
- `assets/css/styles.css`: Estilos globais e responsividade.
- `assets/img/`: Imagens usadas nos artigos.

## Como executar
1. Abra `index.html` no navegador.
2. Navegue pelos links do menu para acessar as outras paginas.

## Observacoes
- Projeto 100% estatico (sem build ou dependencias).
- Conteudo em portugues.
