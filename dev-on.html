Origem

O HTML (HyperText Markup Language) foi criado em 1990-1991 por Tim Berners-Lee, físico do CERN, com o objetivo de facilitar o compartilhamento de documentos entre pesquisadores. Baseado no conceito de hipertexto, a primeira versão apresentava 18 tags básicas, evoluindo para o HTML5 em 2014, que trouxe tags semânticas modernas.

HTML semântico

HTML semântico é o uso de tags específicas como: header, nav, article, footer. Que dão significado ao conteúdo, em vez de apenas definir sua aparência com (div). Essa abordagem melhora o SEO, acessibilidade para leitores de tela e a manutenção do código por desenvolvedores.
